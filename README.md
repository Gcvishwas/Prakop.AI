# Prakop.AI  
**Nepali Disaster Response Chatbot**

Prakop.AI is a Nepali-language disaster response chatbot built to assist people during natural disasters by providing reliable information, guidance, and support in the Nepali language. The system uses modern full-stack web technologies and the Google Gemini API to generate intelligent, context-aware responses.

This project is suitable for academic use (CSIT internship / project) as well as a strong full-stack + AI portfolio project.

---

## Features

- Nepali language disaster-response chatbot  
- AI-powered responses using Google Gemini API  
- User authentication and session management with Clerk  
- Chat history storage for logged-in users  
- Responsive and clean UI using Tailwind CSS
- Emergency contacts
- RESTful backend API  
- Persistent data storage using MongoDB  

---

## Tech Stack

### Frontend
- React.js  
- Tailwind CSS  
- React Router  
- TanStack Query (React Query)  
- Clerk Authentication  

### Backend
- Node.js  
- Express.js  
- MongoDB  
- Google Gemini API
